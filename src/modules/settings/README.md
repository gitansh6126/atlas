# Settings Module

Provides the settings UI for configuring Atlas behavior, appearance, and preferences.

## Responsibility
- Application preferences (theme, language, editor settings)
- Workspace settings (name, description)
- Keyboard shortcuts reference
- Storage management (data size, clear, export)
- Account settings (profile, auth — future)
- Plugin management (enable/disable — future)
- About / version information

## Current Status (Phase 1)
- Basic settings page with theme toggle
- Workspace name display
- Placeholder sections for future settings

## Never
- Import editor or plugin modules
- Handle storage directly (use storage interface)
