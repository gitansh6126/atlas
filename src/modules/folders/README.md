# Folders Module

Manages folder hierarchy for organizing pages within a workspace.

## Responsibility
- Folder CRUD (create, rename, delete)
- Nested folder structure (tree with parent-child)
- Drag-and-drop reordering (future)
- Folder collapse/expand state
- Folder-level permissions (future)

## Current Status (Phase 1)
- Folder tree UI in sidebar
- Collapsible sections
- Context menu (rename, duplicate, delete — placeholder)
- Data model and store integration

## Never
