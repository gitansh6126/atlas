# Media Module

Manages file uploads, images, videos, and other media assets used across pages.

## Future Responsibility
- Media library (browser for all uploaded files in workspace)
- Upload handling (drag-and-drop, file picker, paste)
- Image optimization (resize, compress, format conversion)
- Media transformations (filters, cropping — future)
- Media embedding (images, videos, audio in editor)
- File storage abstraction (local filesystem, S3, Supabase Storage)
- Media metadata (dimensions, file size, format, EXIF)
