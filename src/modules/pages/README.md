# Pages Module

Manages pages — the fundamental content unit in Atlas. Pages belong to folders or exist at workspace root.

## Responsibility
- Page CRUD (create, rename, delete, duplicate)
- Page selection and navigation
- Favorites and recent pages tracking
- Page metadata (created, updated, last opened)
- Page templates (future)

## Current Status (Phase 1)
- Page tree UI in sidebar
- Favorites section
- Recent pages section
- Star/unstar toggle
- Context menu (rename, duplicate, move, delete — placeholder)
- Data model and store integration

## Never
- Render page content (editor handles that)
- Know about editor state or plugins
