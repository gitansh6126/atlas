# Formatting

Atlas supports rich inline formatting to make your text clear and expressive.

## Inline Formatting

| Format | How to Apply | Shortcut |
|--------|-------------|----------|
| **Bold** | `**text**` or `__text__` | `Ctrl + B` |
| *Italic* | `*text*` or `_text_` | `Ctrl + I` |
| <u>Underline</u> | N/A | `Ctrl + U` |
| ~~Strikethrough~~ | `~~text~~` | val |
| `Code` | `` `code` `` | N/A |
| [Link](https://example.com) | `[text](url)` | `Ctrl + K` |

## Markdown Support

Atlas supports common **Markdown** syntax for quick formatting:

```markdown
# Heading 1
## Heading 2
**Bold text**
*Italic text*
[Link text](https://example.com)
`inline code`
```

Type these directly into a paragraph block and Atlas will format them automatically.

## Formatting Tips

- 💡 Use **bold** to highlight key terms or actions
- 💡 Use *italic* for emphasis or foreign words
- 💡 Combine them: `***bold and italic***` → ***bold and italic***
- 💡 Use `code` for file names, commands, or technical terms
