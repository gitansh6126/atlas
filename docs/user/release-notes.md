# Release Notes

This page tracks the latest changes, improvements, and additions to Atlas.

---

## Version History

### v1.0.0 — Initial Release 🎉

- Introduced Atlas, a block-based knowledge workspace
- Support for multiple workspaces
- Core block types: Paragraph, Heading, List, Divider, Grid, Card, Gallery, Kanban, Calendar
- Labels and Tags for container blocks
- HTML Embed with file upload support (`.html`, `.css`, `.txt
- Full-text search across all pages
- Light and Dark themes
- Keyboard shortcuts and slash commands
- Markdown export
