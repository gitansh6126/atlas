# Slash Commands

Slash commands are the fastest way to insert blocks and format content in Atlas. Just press `/` and pick what you need.

## How to Use the `/` Menu

1. Click anywhere on the page
2. Press `/`
3. A menu pops up showing available commands
4. Type the command name (e.g., `heading`) or scroll and click

---

## Available Slash Commands

| Command | What It Does |
|---------|-------------|
| `/heading` | Insert a heading block (H1, H2, H3) |
| `/paragraph` | Insert a plain text paragraph |
| `/list` | Insert a bullet list item |
| `/divider` | Insert a horizontal divider |
| `/grid` | Insert a responsive grid block |
| `/card` | Insert a card container block |
| `/gallery` | Insert a gallery layout block |
| `/kanban` | Insert a kanban board block |
| `/calendar` | Insert a calendar block |
| `/label` | Insert a label badge |
| `/tag` | Insert a colored tag pill |
| `/html` or `/embed` | Insert an HTML embed block |

---

## Pro Tips

- ⏩ **Type faster**: you do not need to type the full command. `/h` will match `/heading`.
- ↑↓ **Navigate with arrows**: Use the **Up** and **Down** arrow keys to select a command, then press **Enter**.
- ⌫ **Dismiss**: Press **Escape** to close the menu without inserting anything.

---

> 💡 **Tip**: You can also transform existing blocks using slash commands. Just place your cursor in a block, press `/`, and choose the new type.
