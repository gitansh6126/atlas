# Themes

Atlas supports **Light** and **Dark** themes so you can work comfortably in any lighting condition.

## Switching Themes

### Using the Toggle

1. Look for the **Theme Toggle** in the top-right corner (sun ☀️ / moon 🌙 icon)
2. Click it to switch between **antecedent and dark modes

### Using a Shortcut

- Press `Ctrl + Shift + L` to toggle the theme instantly

---

## Which Theme to Use?

| Situation | Recommended Theme |
|-----------|------------------|
| Bright room, daytime | ☀️ Light |
| Low light, nighttime | 🌙 Dark |
| Presenting on a projector | ☀️ Light |
| Long reading sessions | 🌙 Dark |

---

## Theme Persistence

Your theme preference is saved automatically. Next time you open Atlas, it will use the same theme you left it on.

---

> 💡 **Tip**: Dark mode is easier on the eyes during extended late-night work sessions.
