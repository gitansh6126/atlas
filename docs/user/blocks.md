# Blocks

Blocks are the building units of every page in Atlas. Every line, container, or element you see is a block. You insert and arrange blocks to build rich documents.

## How to Insert a Block

1. Press `/` anywhere on the page to open the **slash menu**
2. Start typing the block name (e.g., `heading`, `list`)
3. Press **Enter** or click the block to insert it

You can also press **Enter** at the end of a line to create a new block.

---

## Block Types

### 📝 Paragraph
The default block for plain text. Just start typing.

### ✅ List (Bullet List)
Renders as bullet points. Press **Enter** to add another item.

```
- First item
- Second item
```

### 🔜 Heading (H1, H2, H3)
Organizes your page into sections.

- Type `# ` at the start of a line for **Heading 1**
- Type `## ` for **Heading 2**
- Type `### ` for **Heading 3**

### ➖ Divider
A horizontal line to visually separate sections.

### ⊞ Grid
A responsive grid layout to arrange blocks side-by-side. Drag other blocks into a grid to organize content multi-column.

### 📋 List (Container)
A block that acts as a parent for other items. Use it to group related content under a title or label.

### 💾 Card
A visually distinct container with a border and padding. Great for highlighting important notes, quotes, or steps.

### 🎨 Gallery
Displays items in a visual grid, ideal for showcasing images or card previews.

### 📊 Kanban
A board layout with columns and cards, perfect for task management, project tracking, or workflow visualization.

Drag cards between columns to change their status.

### 📅 Calendar
A date-based container that shows items mapped to specific dates, useful for planning, scheduling, and timelines.

### 🏷️ Label
Inline styled badge text. Use labels within containers or inline to highlight statuses or categories.

- Variants: **default**, **secondary**, **outline**, **destructive**
- Example use: marking a card as `Done`, `Draft`, or `Urgent`

### 🐙 Tag
Colored pill-shaped badges with custom hex colors and auto-adjusted text contrast.

- Set any **hex color** (e.g., `#3B82F6`, `#FF5733`)
- Text color is chosen automatically for readability
- Great for categorizing and color-coding your content

### 📤 HTML Embed
A powerful block that lets you embed custom **HTML** and **CSS**.

- Supports **drag & drop** or **click to upload** `.html`, `.css`, `.txt` files
- Edit in separate tabs: **Preview**, **HTML**, **CSS**, **Files**
- Perfect for custom widgets, embedded charts, or styled snippets
