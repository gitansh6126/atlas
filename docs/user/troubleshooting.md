# Troubleshooting

If something is not working in Atlas, this guide will help you fix common issues quickly.

## Atlas Won't Open

1. **Check your OS**: Make sure you downloaded the correct version for your operating system.
2. **Restart your computer** and try launching Atlas again.
3. **Check for updates**: You might be on an older version. Download the latest release.

---

## Workspace Not Found or Missing

1. Make sure the **workspace folder** on your computer was not moved or renamed.
2. Re-open Atlas and point it to the correct folder location.
3. If the folder is gone, check your computer's **Recycle Bin** or **Trash**.

--- 

## Search is Not Finding Results

1. Make sure you are in the correct workspace.
2. Try re-indexing search by restarting Atlas.
3. Check that the page has been saved (Atlas auto-saves, but unsaved drafts may not appear).

---

## Sidebar is Stuck or Hidden

1. Press `Ctrl + B` (or `Cmd + B`) to toggle the sidebar visibility.
2. If it is collapsed, click the arrow icon to expand it.
3. If the width is wrong, switch to **Variable Mode** and drag the handle to reset the size.

---

## Block is Not Rendering Correctly

1. **HTML Embed blocks**: Check the HTML and CSS tabs for syntax errors.
2. **Container blocks** (Grid, Card, etc.): Make sure child blocks are properly nested.
3. Restart Atlas as a last resort.

---

## Reporting a Bug

If none of the above helps, please report the issue with details:

- What you were trying to do
- What you expected to happen
- What actually happened
- Your operating system and Atlas version
