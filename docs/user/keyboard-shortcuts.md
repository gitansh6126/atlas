# Keyboard Shortcuts

Master these shortcuts to speed up your workflow in Atlas.

## Text Formatting

| Shortcut | Action |
|----------|--------|
| `Ctrl + B` | Bold |
| `Ctrl + I` | Italic |
| `Ctrl + U` | Underline |
| `Ctrl + K` | Insert or edit link |
| `Ctrl + Shift + K` | Remove link |

## Block Management

| Shortcut | Action |
|----------|--------|
| `/` | Open slash command menu |
| `Enter` | Create new block below |
| `Shift + Enter` | Insert line break within block |
| `Ctrl + Shift + D` | Delete current block |
| `Ctrl + Shift +MUp` | Move block up |
| `Ctrl + Shift + Down` | Move block down |

## Page Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Create new page |
| `Ctrl + E` | Rename current page |
| `Ctrl + Delete` | Delete current page |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + F` | Open search |
| `Ctrl + [` | Go back |
| `Ctrl + ]` | Go forward |

## View & Interface

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + L` | Toggle light/dark theme |
| `Ctrl + B` | Toggle sidebar (macOS: `Cmd + B`) |

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Ctrl + S` | Save (Atlas auto-saves, but this forces it) |

---

> 💡 **Tip**: On macOS, replace `Ctrl` with `Cmd` where applicable.
